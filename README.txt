本项目是C语言大作业
1、本文件是说明文件。
2、taleCCompiler.c是我们主要编写的，针对自行设计的编程语言taleC的编译器（基于C编译器）。
3、taleCBuilder.bat是一键生成最终可执行的taleC语言编写的程序的批处理文件（需要gcc环境，确保此bat文件与taleCCompiler.c，taleC.tc在同一目录下）。
4、taleC.tc是存放taleC语言代码的文件，里面写着我们的示例程序（可用记事本打开，本质上是个txt）。
5、taleC_Language_Rule.md是我们自行设计的编程语言的语言规范简介。

您可将bat文件中的gcc修改为您电脑上对应的C编译器。
